Описание проекта
================

Система управления складами предназначена для учета, контроля и управления товарными запасами на складах. 
Программа позволяет вести учет товаров, поставщиков, складов и контролировать перемещение товаров.

Основные возможности:
---------------------

* Управление складами: добавление, редактирование и удаление информации о складах
* Управление запасами: контроль наличия товаров на складах
* Работа с поставщиками: ведение базы данных поставщиков
* Управление заказами: оформление и отслеживание заказов
* Учет продуктов: добавление и редактирование информации о продуктах
* Экспорт данных: выгрузка отчетов в различных форматах (Excel, CSV, PDF)
